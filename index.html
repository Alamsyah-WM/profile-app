Robot Arm Pemilah Plastik Kresek dan Botol Plastik

Robot Arm akan mendeteksi suatu objek terlebih dahulu dengan Webcam yang terletak di atas menembak kearah bawah untuk mendeteksi apakah objek yang berada di daerah Webcam tersebut adalah plastik kresek atau botol plastik. Ketika Webcam mendeteksi adanya plastik kresek atau botol plastik maka Arm Robot akan bergerak mengambil Plastik kresek atau botol plastik tersebut lalu akan diangkat ke belakang dan di buang sesuai tempat sampahnya.

Pusaka

Pusaka merupakan sebuah Website untuk meminjam e-book perpustakaan. User akan diminta membuat akun terlebih dahulu, kemudian user bisa memilih buku berdasarkan genre ataupun trend dan sebagainya. ketika user sudah memilih e-book maka akan ada opsi untuk menentukan berapa lama peminjaman e-book dan akan diberikan link untuk membaca e-book tersebut.

NeverDelay

NeverDelay merupakan aplikasi mobile yang berfungsi untuk memesan tiket bus. NeverDelay bukan hanya tentang memesan tiket, tetapi memberikan pengalaman perjalanan yang optimal bagi setiap user. Disini user akan diminta membuat akun terlebih dahulu, kemudian user dapat memilih lokasi Halte-bus, tujuan dan merek bus. Setelah itu user akan mengatur jumlah penumpang dan memilih jam. Langkah terakhir akan ada barcode untuk melakukan transaksi.
